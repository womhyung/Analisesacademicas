# 📊 Aplicação Acadêmica - Dark Mode

Este projeto é uma aplicação acadêmica em **dark mode**, com navegação entre páginas, feed de análises, modal com gráficos e nuvem de palavras.  
O frontend é publicado via **GitHub Pages** e o backend pode ser hospedado em **Render** com banco de dados no **MongoDB Atlas**.

---

## 🚀 Funcionalidades

- Navegação entre páginas: Home, Login, Feed, Perfil.
- Feed estilo Instagram com **cards reutilizáveis**.
- Modal com **gráfico Chart.js** e **nuvem de palavras**.
- Tema **dark mode** aplicado em toda a interface.
- Estrutura modular com **components** e **pages**.

## ⚙️ Como rodar o frontend no GitHub Pages

2. Adicione os arquivos do frontend na pasta `/frontend`.
